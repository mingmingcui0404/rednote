# Rednote Content Workflow

This repository collects Xiaohongshu/Rednote content assets and Codex workflow templates.

## Contents

- `docs/Codex入门小红书5篇草稿.md`
  - A 5-post Xiaohongshu draft series for Codex beginners.
- `docs/Codex工作流模板包.md`
  - AGENTS.md templates, Skill templates, advanced prompts, a Xiaohongshu workflow case, and a pre-publish checklist.
- `docs/Codex高阶使用手册-从会用到会搭工作流.md`
  - A long-form Codex workflow handbook that can be turned into Xiaohongshu series posts, a PDF guide, or a course outline.
- `skills/xhs-chrome-publisher/SKILL.md`
  - A reusable Codex skill for generating Xiaohongshu-style posts and preparing them in a logged-in Chrome session.

## Safety Notes

The Xiaohongshu publishing skill is designed to prepare drafts, not to perform artificial engagement.

It must not:

- Read, export, or print cookies/tokens.
- Like, favorite, comment, follow, or simulate engagement.
- Bypass CAPTCHA, platform limits, or safety checks.
- Publish publicly without explicit user confirmation.
